# mangofp-front Messaging front - end of MangoFP
For full project setup see readme of mangofp project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files

npm run lint
